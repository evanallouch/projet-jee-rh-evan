# Projet JEE RH / Départements / Projets / Paie

## Objectif

Application web JEE permettant de gérer :
- les employés
- les départements
- les projets
- les fiches de paie

## Stack technique prévue

- Java / JEE (Jakarta EE)
- Tomcat
- JSP / Servlets (pattern MVC)
- JDBC ou JPA / Hibernate pour l’accès aux données
- Base de données relationnelle (MySQL / PostgreSQL)

## Plan de travail

1. Modélisation (MCD, diagramme de classes).
2. Création de la base de données.
3. Entités Java (Employe, Departement, Projet, FichePaie, etc.).
4. DAO / couche d’accès aux données.
5. Servlets + JSP pour les écrans :
   - liste des employés
   - gestion des départements / projets
   - visualisation des fiches de paie
6. Tests et finalisation.
